Tier0의 digital twin 기능을 사용하면 실제와 같은 3D 장면과 모델에서 UNS 데이터를 실시간으로 볼 수 있습니다.

## 워크숍 Digital Twin
이 섹션에서는 워크숍을 실시간으로 표현합니다. 포함된 장비는 다음과 같습니다:
- 선반 2대
- 두 선반 사이에서 작업하는 로봇암 1대
- 인바운드 버퍼 스테이션 1개와 아웃바운드 버퍼 1개
- 품질 검사 장비 1대
- 통합 제어 시스템 1개
- 원자재 선반 2개와 제품 선반 2개
- 1 AGV

## 워크숍 데이터 수집
기계 동작을 제어하는 데이터, 기계 상태를 나타내는 데이터, 워크숍 환경 데이터를 포함한 모든 데이터를 **UNS**로 수집합니다.

2. 모델을 업로드하고 UNS 데이터를 매핑할 모델 노드를 선택합니다.
3. 인스턴스를 만들고 노드 매핑에 사용할 UNS topic을 선택한 뒤 실제 세계 기준 모델 높이를 설정합니다.
4. 필요에 따라 topic 필드를 선택한 모델 노드의 **Position** 및 **Rotation** 각 축에 바인딩합니다.

        하나의 모델에 여러 인스턴스를 만들어 서로 다른 데이터를 가진 동일 장비 라인을 시뮬레이션할 수 있습니다.
        :::

</Steps>

2. **Instance Library**에서 워크숍에 필요한 모든 인스턴스를 추가합니다.
3. 인스턴스 모델을 워크숍 내 위치로 이동합니다.
        :::tip[모델을 올바른 위치로 이동하는 방법]
        - 상단 도구를 전환하여 모델을 이동, 확대/축소, 회전하고 모델 보기 시점과 좌표 위치를 변경합니다.
        - 오른쪽의 **Object**를 확인합니다. 여기에 바인딩된 UNS 모델이 표시되어 모델 위치를 확인할 수 있습니다.
        :::
4. 모델 하나를 선택하고 viewport와 lighting을 설정합니다.
5. 설정이 완료되면 이후 표시를 위해 장면을 저장합니다.
</Steps>

## 외부 공유
<Steps>
1. **Model**로 이동하여 워크숍에서 구성된 모델을 선택합니다.
2. 기존 인스턴스를 선택하고 QR 코드를 생성합니다.
3. 공유 모드를 선택하고 코드를 외부 관계자에게 보냅니다.
        - **Share Model**: 실시간 UNS 데이터가 포함된 모델을 표시하는 Tier0 플랫폼 링크를 생성합니다.
        - **Share Scene**: 모델과 해당 모델이 위치한 장면, 실시간 UNS 데이터 및 환경 설정을 표시하는 Tier0 플랫폼 링크를 생성합니다.
</Steps>
