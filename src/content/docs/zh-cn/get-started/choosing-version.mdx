## Edge 硬件要求

:::tip[如果你想使用 Edge]
Edge 面向技术评估，需要一定的运维经验。
使用前请确保你的硬件满足以下要求。
:::

| | 最低配置 | 推荐配置 |
|---|---|---|
| CPU | 4 cores | 8 cores |
| Memory | 8 GB | 16 GB |
| Disk | 100 GB (1000 IOPS) | 1 TB |
| OS | Ubuntu 24.04、Windows 10/11 (Docker) | - |
