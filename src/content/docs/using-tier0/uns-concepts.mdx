The Unified Namespace (UNS) is Tier0's single source of truth: a semantic tree of paths whose leaves — **topics** — carry live, structured values over MQTT pub/sub.

```mermaid
flowchart TD
  plant["Plant/"] --> line1["Line1/"]
  line1 --> metric["Metric/"]
  line1 --> state["State/"]
  line1 --> action["Action/"]
  metric --> temp(["Temperature"])
  state --> status(["MachineStatus"])
  action --> start(["StartBatch"])

  class temp,status,start t0accent
```
