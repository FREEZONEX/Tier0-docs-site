## What is a Tier0 App

A Tier0 application, is a **container** that integrates with **UNS data foundation**, 
relying on **Event Flow** to build internal logic, and **AI Agent** to combine
them together and provide you a complete solution.

```mermaid
flowchart TB
  subgraph tier0App["Tier0 App"]
    direction TB
    ai["AI Agent<br/>builds and orchestrates"]
    ui["App UI<br/>central experience"]

    subgraph foundation["Data and logic foundation"]
      direction LR
      uns[("UNS<br/>data foundation")]
      ef["Event Flow<br/>backend logic"]
    end
  end

  ai -. "design / operate" .-> ui
  ai -. "read / write" .-> uns
  ai -. "compose logic" .-> ef

  uns -- "display data" --> ui
  ui -- "write back" --> uns
  uns -- "data changes" --> ef
  ef -- "logic result" --> ui

  class tier0App,foundation t0Frame
  class uns t0accent
  class ef t0soft
  class ai t0agent
  class ui t0ui
```
