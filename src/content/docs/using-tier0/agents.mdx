Tier0 is agent-ready by design: the same CLI humans use is packaged as an **agent skill**, and these docs ship machine-readable endpoints.

## Install the Tier0 Skill

```bash
npx @tier0/cli@latest install        # CLI + skills in one step
# or skills only:
npx skills add FREEZONEX/Tier0-skill
```
